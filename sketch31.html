Genuary 31 - Plinko Music

What a journey! I had a wonderful Gibber setup, but the audio context wouldn't set up properly, so I told a deep dive into p5.sound.

Sections of sound are generated and played using noise to move up and down through the generated measures. Reverb is used to create an echo effect.

For extra fun, a plinko board is set up using matter.js to drop balls onto a visualizer. The FFT is used to create force to bounce the balls around.
